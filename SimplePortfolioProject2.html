Project 1: Sports and Fitness Tracker Website

Description:

This is a web-based platform focused on fitness and sports for players, parents, and coaches. The website includes a login system for different user types: players, parents, and coaches. It helps users track their exercises and sports activities, and it shows nearby sports grounds using location-based suggestions. The platform also provides personalized meal recommendations for athletes, workout routines, and access to real coaches for better training. There's also a feature for premium players to get advanced resources and support for better performance.

Key Features:
  • Login interface for players, parents, and coaches
  • Activity tracking and sport history
  • GPS-based ground locator
  • Meal and nutrition suggestions
  • Access to real certified coaches
  • Premium section for high-level athletes

Project 2: FlexiCart – Smart Shopping Experience

Description:

FlexiCart is a basic online shopping website designed for a smooth and flexible shopping experience. It allows users to browse products, view product details, and add items to the cart. The website layout is built using clean HTML and basic CSS to keep the interface user-friendly and responsive. This project helped me understand how e-commerce websites work at a basic level, including product display, cart system layout, and category-wise organization.

Key Features:
  • Simple product listing with categories
  • Product description and images
  • Add-to-cart feature layout
  • Clean and user-friendly interface

Project 3: College Portal – Attendance and Academic Info

Description:

This is a basic college portal web page created to help students and faculty access essential academic details. The portal includes features such as attendance tracking, subject-wise lecture notes, teacher contact information, timetables, and previous year’s question papers (PYQs). It’s built using only HTML and CSS and is designed to simulate the structure of a real university dashboard with student-focused features.

Key Features:
  • Attendance tracking view
  • Access to notes and recorded lectures
  • Teacher directory and contact info
  • Timetable and event calendar
  • Previous year question paper section
